# web-app

Language target: TypeScript (Next.js).

## Responsibility

- Exploration UX: graph, impact, lineage, evidence-backed query results.
- Governance UX: reviewer actions, approval gates, auditability.
- Migration readiness dashboard with gate status and blockers.

## Key UX Constraints

- Rejected and low-support claims remain visible.
- Block migration-impacting actions on rejected/low-support claims.
- Sandbox artifacts must be marked non-actionable.
